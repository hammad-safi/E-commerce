## Order Tracking Flow

```
CUSTOMER WANTS TO TRACK:

1. Go to /track-order page

2. Enter:
   - Order ID (e.g., ORD-20260121-1234)
   - Phone Number (used during checkout)

3. System Queries:
   - Find order by ID AND Phone
   - Security: Prevents unauthorized access
   
4. Display:
   - Order Status (Pending/Processing/Shipped/Delivered)
   - Payment Status
   - Items Ordered
   - Delivery Address
   - Timeline
```
